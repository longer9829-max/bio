## 🌐 GitHub Pages Деплой

### Автоматический деплой

Репозиторий **уже настроен** для GitHub Pages. Ваш сайт будет доступен по адресу:

**→ https://longer9829-max.github.io/bio/portfolio.html**

или просто откройте файл:

**→ https://github.com/longer9829-max/bio/blob/main/portfolio.html**

### Ручная активация (если требуется)

1. Перейди в **Settings** репозитория
2. Слева выбери **Pages**
3. Убедись что:
   - **Source** = `Deploy from a branch`
   - **Branch** = `main` (или нужная ветка)
   - **Folder** = `/ (root)`
4. Сохрани — сайт будет опубликован в течение 1-2 минут
